Recommendation System with Item-Item Collaborative Filtering

A recommender system prototype based on item-item collaborative filtering

Download as .zip Download as .tar.gz View on GitHub

Setup

The system is built with LensKit, an open-source took kit for building recommenders.

Requires the following:

Java SE Development Kit 7

LensKit

Apache Maven

Java IDE such as Eclipse or IntelliJ IDEA.

Background

This recommendation system prototype uses item-item collaborative filtering. Here for details.

The algorithm is composed of the following steps.

  1. Building a model by computing similarities between items. We do this by: a. Subtract the user's mean rating from each rating prior to computing similarities. b. Calculate cosine similarity between items.

  2. Score the items. By using a pre-defined number of neighbours computed in step 1, the item is scored using weighted sum.

  3. The item-item similarity matrix is also useful for 'find similar items' features. Items can be scored with the sum of its similarity to each of their reference items.

Test Drive

A set of test data is provided for movie ratings, but can be easily adopted for other domains.

data/movie-titles.csv Maps Movie IDs to Movie Titles.

data/ratings.csv Users and their movie ratings. Each line of the CSV file is ordered as: User ID, Movie ID, Rating

data/users.csv Maps User ID to User Name.

Run the recommender with command similar to the following: run-ii 1024:77 1024:268 1024:462 1024:393 1024:36955 2048:77 2048:36955 2048:788

For each pair, the first item is the User ID and the second pair is the Movie ID.

Outputs associate score for each move/user pair.
